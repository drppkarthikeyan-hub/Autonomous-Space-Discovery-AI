# Autonomous Space Discovery AI

## Project Overview
This project builds an autonomous AI system capable of identifying galaxies, stars, exoplanets, and unknown space anomalies using astronomical datasets.

## Research Goal
Develop an AI-driven scientific discovery platform that can classify known celestial objects and detect unknown anomalies for future space exploration.

## Dataset Sources
- NASA Open Data
- Sloan Digital Sky Survey (SDSS)
- ESA Space Data Archive

## Core Components
- Data Collection
- Data Preprocessing
- Deep Learning Classification
- Anomaly Detection
- Research Visualization
- Cloud Deployment

## Technologies
- Python
- PyTorch
- TensorFlow
- OpenCV
- Scikit-learn
- SQL
- Google Cloud
